# 一期商城实战

## 商业模式（CEO的视野）

不管任何行业，不一定是互联网项目，先解决的是商业模式的问题：如下3个问题

智商

情商 “商业模式”

1、客户从哪来
2、怎么让他成交
3、怎么让他重复成交

抓潜 - 成交 - 鱼塘

## 需求模式（产品经理视野）

产品经理的关注点：怎么实现老板的商业模式

   抓潜 ———————————————— 成交 ———————————————— 鱼塘
(注册，活动)     (商品维护、下订单、活动)     (相关推荐、活动)
